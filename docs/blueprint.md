# **App Name**: CoinSender

## Core Features:

- Username Validation: Validate recipient usernames in real-time to ensure they exist and are correctly formatted (starts with @).
- Coin Package Selection: Display a grid of coin packages with options like 30, 350, 700, 1,400, 3,500, 7,000, and 17,500 coins, including a 'Custom' option for users to input a specific amount.
- Live Total Calculation: Automatically calculate the total amount in coins and currency based on the selected package or custom amount.
- Send Coins Transaction: Enable users to send coins to a specified recipient after validating the username and selecting a coin package.
- Error Handling and Notifications: Display error messages for invalid usernames, insufficient balance, or failed transactions.
- Simulated User Lookup: A tool is used to confirm the existence of the username. Generative AI can conditionally use this to show a preview or proceed.

## Style Guidelines:

- Primary color: Soft pink (#F472B6) for CTAs and highlights, aligning with the requested aesthetic of luxury and approachability.
- Background color: Light neutral off-white (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Muted lavender (#A855BA) to add depth and a touch of sophistication.
- Body and headline font: 'Poppins' (sans-serif) for a clean, readable, and contemporary style, matching the minimal and bold aesthetic.
- Use simple, modern coin icons with a slight 3D effect to enhance the luxurious feel.
- Mobile-first approach with a centered card layout, ensuring large touch targets and stacked elements for ease of use on smaller screens.
- Incorporate subtle animations on hover and selection, such as a gentle scale effect and a check icon appearance for selected coin packages.
